# WhaleBell 分销系统 / Sistem Distribusi WhaleBell

---

## 🇨🇳 中文版

### 什么是分销？
你邀请其他主播注册 WhaleBell，她们买 VIP 时你就能赚佣金。

### 收益级别

| 级别 | 说明 | 收益 |
|:--:|------|------|
| L1 直推 | 你用邀请码拉来的主播 | 她买 VIP，你拿 **40%** |
| L2 间推 | 你的直推再拉来的人 | 那人买 VIP，你拿 **10%** |

### 价格和佣金

| 套餐 | 价格 | L1 佣金 | L2 佣金 |
|------|------|:--:|:--:|
| 周卡 | 29,000 IDR | 11,600 | 2,900 |
| 月卡 | 79,000 IDR | 31,600 | 7,900 |

### 操作步骤

1. 打开 WhaleBell → 点底部 **邀请+2**
2. 系统生成你的专属 6 位邀请码
3. 复制链接发给姐妹（WhatsApp / TikTok / 微信群）
4. 姐妹用你的码注册 → 双方各 +2 大哥名额
5. 姐妹买 VIP → 你自动获得佣金
6. 佣金记账在系统里，月底统一结算

### 额外奖励

- 邀请满 **2 人** → 解锁全部 VIP 大哥
- 邀请满 **5 人** → 永久免费 VIP
- **拼团**：拉 3 个姐妹一起 → 全团解锁 VIP

## 🇮🇩 Versi Bahasa Indonesia

### Apa itu Distribusi?
Kamu mengajak streamer lain daftar WhaleBell. Saat mereka beli VIP, kamu dapat komisi.

### Level Penghasilan

| Level | Penjelasan | Penghasilan |
|:--:|------|------|
| L1 Langsung | Streamer yang daftar pakai kode kamu | Dia beli VIP, kamu dapat **40%** |
| L2 Tidak Langsung | Temannya streamer L1 kamu | Orang itu beli VIP, kamu dapat **10%** |

### Harga & Komisi

| Paket | Harga | Komisi L1 | Komisi L2 |
|------|------|:--:|:--:|
| Mingguan | 29.000 IDR | 11.600 | 2.900 |
| Bulanan | 79.000 IDR | 31.600 | 7.900 |

### Cara Kerja

1. Buka WhaleBell → klik **Ajak +2** di bawah
2. Sistem generate kode undangan 6 huruf
3. Copy link dan kirim ke teman (WhatsApp / TikTok / grup)
4. Teman daftar pakai kode kamu → kalian berdua dapat +2 target
5. Teman beli VIP → kamu otomatis dapat komisi
6. Komisi dicatat di sistem, dibayarkan setiap akhir bulan

### Bonus Tambahan

- Ajak **2 orang** → buka semua target VIP (gratis)
- Ajak **5 orang** → VIP gratis selamanya
- **Grup Beli**: Ajak 3 teman barengan → seluruh grup buka VIP

## Contoh / 例子 / Contoh Kasus

```
Kamu (A) mengajak Sari
  -> Sari daftar: A dan Sari masing-masing +2 kuota
  -> Sari beli Bulanan (79K): A dapat 31,600 IDR

Sari mengajak Dewi
  -> Dewi daftar: Sari dan Dewi masing-masing +2 kuota
  -> Dewi beli Bulanan (79K):
      Sari dapat 31,600 IDR (L1)
      A dapat 7,900 IDR (L2)

Total penghasilan A: 31,600 + 7,900 = 39,500 IDR
```
